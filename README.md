# 🌸 Iris Flower Classification

An end-to-end Machine Learning project that classifies Iris flower species (Setosa, Versicolor, and Virginica) using Logistic Regression. The project includes an interactive Streamlit web application for real-time predictions based on flower measurements.

---

## 📌 Features

- Predicts Iris flower species
- Interactive Streamlit web application
- Trained using Logistic Regression
- User-friendly interface
- Fast and accurate predictions

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit
- Pickle

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Iris-Flower-Classification.git
```

Go to the project folder:

```bash
cd Iris-Flower-Classification
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

or

```bash
python -m streamlit run app.py
```

---

## 📊 Input Features

- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

---

## 🎯 Output

The model predicts one of the following Iris flower species:

- Setosa
- Versicolor
- Virginica

---



## 📈 Model

- Algorithm: Logistic Regression
- Library: Scikit-learn
